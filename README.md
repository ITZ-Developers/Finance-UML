## Finance UML

Dự án mô hình hóa UML cho hệ thống tài chính

## Mô tả
Dự án này chứa các mô hình UML để thiết kế và phân tích hệ thống tài chính, bao gồm các diagram như:
- Use Case Diagram
- Class Diagram  
- Sequence Diagram
- Activity Diagram
- State Diagram

## Công nghệ sử dụng
- PlantUML
- Draw.io
- Visual Paradigm
